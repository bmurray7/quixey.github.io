Quixey iOS DeepLink SDK

Overview

The Quixey iOS Router SDK is a set of libraries that allow iOS developers to enable Quixey Deep Link and BCT capabilities to their applications.

Prerequisites

  • iOS 8+
  • Xcode

App Registration Required. Please contact Quixey directly to obtain your Quixey App ID. You will need to provide your bundle identifier and application version.

Installation Steps

  1. Download the latest SDK here.

  2. Drag and drop QuixeySDK.framework into Xcode project. Check the Copy Items if needed option in the dialog and click Finish.

  3. Select the project file in the left pane.

    • Under General tab check if the sdk is listed in the Embedded Binaries section. If not, click the plus (+) button and select QuixeySDK.framework.
    • Under Info tab navigate to the URL Types section. Click the plus (+) button and set value for URL Schemes as qx{QUIXEY_APP_ID} with your Quixey app id and the prefix qx. Example: qxyelpapp

  4. In application:didFinishLaunchingWithOptions launchOptions: add these following lines of code

    UIApplication.registerSelectorToPopBackToHome(){
   // Code to get back to home screen
}

  5. Don't forget to import the umbrella header #import <QuixeySDK/QuixeySDK.h>

  6. Build your app to make sure that SDK was installed correctly.

Widgets/Actions Covered

  • UITableViewCell
  • UICollectionViewCell
  • UIButton
  • UITextField
  • UIControl
  • UISearchBar
  • UIView with a tap gesture recognizer
  • Segue

Error alerts

In case of errors during replay, SDK will throw alert stating the reason for failure. Possible error situations are

  • Invalid/Outdated BCTs
  • Unable to parse the input from search application
  • Unable to download the BCTs from server
  • Unable to perform replay
